Recent activity

The recent activity let you keep track of the testing activity of your organization. [Ça reprend un peu ce qu'il y a sur la home page.]

  • See the summary of passed/failed tests
  • See who has run the tests
  • See which projects and versions are gathered in a report
  • See the categories of tests

When you are browsing to a project, you can track the recent activity of the specific project. [L'idée c'est de dire qu'on a ces infos aussi spécifiquement pour un projet particulier.]

Test repartition

See how are distributed your tests across categories. [Expliquer qu'on peut sortir la répartition des tests par catégories.]

You can see this repartition across your organization or for each of your project. [Expliquer qu'on peut voir soit pour tout soit pour un projet spécifique.]

Test contributors

Keep track of people who have made contributions to your organization's tests. [Expliquer qu'on garde une trace de qui participe à l'activité de tests]

As for the other features, you can get this information by project. [Comme d'hab, on peut le faire sur un projet particulier.]

Test time execution

Understanding why a test is taking more and more time to run is important to detect code issues early in the development process.[Expliquer le contexte de la feature]

The trend of test execution (red line) give you this information allowing you to investigate why the test is becoming slower and slower.[Expliquer à quoi sert le widget et ce qu'il permet]
