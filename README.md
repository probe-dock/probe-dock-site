# Probe Dock Site

Jekyll site for Probe Dock.

## Requirements

* Ruby 2.x

## First-time setup

* `git clone git@github.com:probedock/probedock.github.io.git`

* `cd probedock.github.io.git`

* `git checkout source`

* `bundle install`

## Serve locally

* `jekyll serve`

## Build from current directory and push to GitHub Pages

* `bundle exec rake pages`
